📖 使用說明

快速上手 WeaMind 天氣查詢

🚀 快速開始

💬 文字查詢

直接在聊天室輸入地點名稱，例如：「台北」、「台中市」、「新竹縣竹北市」

📍 地圖查詢

點擊聊天室左下角的「+」按鈕，選擇「位置資訊」，在地圖上點選任意位置即可查詢該地天氣

⚡ Rich Menu 快速操作

使用下方選單快速查詢常用地點的天氣，或設定個人偏好

🎯 Rich Menu 功能

🏠 居住地天氣

設定並快速查詢您的居住地天氣資訊

🏢 工作地天氣

設定並快速查詢您的工作地天氣資訊

⚙️ 個人設定

管理您的個人偏好設定，包括居住地、工作地等資訊

🕐 最近查詢

快速重新查詢最近搜尋過的地點天氣

📋 其它功能

系統公告、版本更新、使用說明等更多資訊

📝 使用步驟

  1. 加入 WeaMind 為好友
  2. 點擊下方 Rich Menu 或直接輸入地點名稱
  3. 首次使用建議先設定居住地和工作地
  4. 享受快速便利的天氣查詢服務
💡 小提示
• 支援台灣各縣市、鄉鎮區的天氣查詢
• 可使用中文地名，如「台北」、「高雄市」等
• 地圖查詢可選擇台灣任意位置，無需輸入文字
• 設定常用地點後可快速查詢，節省時間
